# VerseMentor

Cross-platform poem recitation app. This repo is organized as a monorepo:

- `shared-core/` TypeScript core logic (ArkTS compatible)
- `android/` Android shell (placeholder)
- `harmonyos/` HarmonyOS shell (placeholder)
- `harmony-next/` HarmonyOS NEXT shell (placeholder)
- `docs/` Build and run notes
